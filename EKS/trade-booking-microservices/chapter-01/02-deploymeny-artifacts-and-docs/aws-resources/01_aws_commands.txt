
1. Connect to eks cluster
    aws eks --region ap-south-1 update-kubeconfig --name amit-modi

    cluster name : scrumptious-jazz-sparrow 

2. See all nodes in the cluster
    kubectl get nodes


3. Add a VPC network plugin if node group does not have it already
    aws eks create-addon --cluster-name amit-modi  --addon-name vpc-cni
    Check for VPC creation: kubectl get pods -n kube-system

    Avoid overriding
    aws eks create-addon --cluster-name amit-modi --addon-name vpc-cni --resolve-conflicts OVERWRITE

    Delete : kubectl delete daemonset aws-node -n kube-system

    Update VPC CNI plugin
    aws eks update-addon --cluster-name amit-modi --addon-name vpc-cni --resolve-conflicts OVERWRITE

    How to delete add-on
    aws eks delete-addon --cluster-name amit-modi --addon-name vpc-cni

    kubectl get pods -n kube-system -l k8s-app=aws-node

4. Install CRDs
    kubectl apply -f https://raw.githubusercontent.com/kubernetes-sigs/aws-load-balancer-controller/main/config/crd/bases/elbv2.k8s.aws_targetgroupbindings.yaml

5.  Install kube-proxy
    aws eks create-addon --cluster-name amit-modi --addon-name kube-proxy

6. Install CoreDNS
    aws eks create-addon --cluster-name amit-modi --addon-name coredns

7. Delete Pods 
    kubectl delete pod -n trading-dev-aws -l app=refdata-provider-aws

8. Describe a pod for error 
    kubectl describe pod -n kube-system aws-node-gsrvp

9. Check Fluent is installed
    kubectl get pods -n amazon-cloudwatch
